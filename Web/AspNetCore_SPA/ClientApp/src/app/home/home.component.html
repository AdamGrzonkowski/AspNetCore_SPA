AspNetCore_SPA

Welcome to single-page application, built with:

  • ASP.NET Core 2.1 and C# for cross-platform server-side code
  • Angular 6.1.1 and TypeScript for client-side code
  • Bootstrap for layout and styling
  • Swagger for documenting
  • Entity Framework Core 2.1 for data access (in-memory)

Application is a simple To-Do app, more like a Proof of Concept, rather than real-world app.
